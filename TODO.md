## Known Bugs From This Update

* The "Update All" button has the wrong kind of tooltip.
* Importing a mod as a directory always triggers the contingency code for mods that have no metadata.
* The program cannot launch the game if the Steam Workshop folder does not exist. This can be solved by installing a mod from the Workshop, then uninstalling it.
* If a workshop image fails to download, it will be permanently stuck with the "Error! :(" image, because that is what gets saved to the cache.
* Refreshing the mod list will cause disabled mods to stop being partly transparent until reopened.
